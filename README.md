# Volunteer Event Management App

   Flutter app untuk manajemen event volunteer dengan fitur:
   - Login/Register user
   - Buat dan kelola event
   - Sistem kehadiran volunteer
   - Generate sertifikat
   - Laporan event
   - Dashboard admin
